1. Admin masuk ke aplikasi
2. Admin akan mengisi data untuk login
3. JIKA admin belum punya akun silahkan isi data di register JIKA TIDAK silahkan Login
4. JIKA admin mengisi data dengan benar silahkan masuk ke dashboard Jika tidak isi data lagi dengan benar
5. tampilan dashboard sudah ditampilkan 
6. Admin akan melihat data-data member yang sudah jatuh tempo
7. JIKA masa berlaku member 1 hari sebelum habis maka beritahu admin kalau member ini masa berlakunya sudah habis JIKA member sudah jatuh tempo maka beritahu admin kalau member ini masa berlakunya sudah habis JIKA member sudah jatuh tempo selama 3 bulan maka member sudah tidak aktif dan dikenakan registrasi ulang 
8. JIKA ada member baru maka Admin akan segera mencantumkan data-nya
9. JIKA member sudah bayar maka admin akan mengubah masa berlakunya
